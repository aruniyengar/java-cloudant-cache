This project provides a {@link com.cloudant.client.cache.CacheWithLifetimes} that can be used with {@link com.cloudant.client.cache.DatabaseCacheWithLifetimes} to provide a cache for Cloudant databases used with the java-cloudant.

The {@link com.cloudant.client.cache.redis.RedisCache} uses Redis for the cache storage.

For additional information see the javadoc for cloudant-client-cache .
